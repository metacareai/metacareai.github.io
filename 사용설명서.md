## 앱 개요

스마트 메타케어는 암환자 및 건강 관리 사용자를 위한 PWA(Progressive Web App)입니다.  
식단 사진 분석, 운동 기록, 증상·복약 추적, AI 피드백을 제공합니다.

---

## 화면 구성

### 1. 랜딩 화면 (로그인 전)
- 앱 소개 카드 3개: 📸 식단 분석 / 🏃 운동 기록 / 💬 AI 코치
- 이용 방법 3단계 안내
- **로그인**: 이름 + 출생년도 입력
- **회원가입**: 신규 사용자 등록

### 2. 홈 화면 (로그인 후)
- **달성현황 칩**: 아침·점심·저녁·운동·컨디션·증상·복약 — 각 칩 터치 시 해당 기록 화면으로 이동
- **오늘의 식단 분석**: AI 분석 결과 접기/펼치기 가능
- 식사 기록, 걸음수, 운동 기록 빠른 입력

### 3. 식단 화면
- 아침/점심/저녁 사진 촬영 또는 메모 직접 입력
- **메모 우선 분석**: 메모가 있으면 사진보다 메모 내용 기준으로 AI 분석
- **회복식단** 모드 (암환자): 항암 치료 중 적합한 식단 평가
- 식단 재분석 기능

### 4. 운동 화면
- 걸음수 입력
- 운동 종류·시간 기록

### 5. 증상·복약 화면 (암환자 전용)
- 통증/피로/배뇨 증상 점수 기록
- 복약 목록 관리
- PSA 수치 기록

### 6. AI 코치 화면
- 식단·운동·증상 종합 AI 피드백

## 어드민 기능

### 어드민 진입 방법
1. 화면 상단 아무 곳이나 **5번 빠르게 탭**
2. 비밀번호 입력 → 어드민 화면 진입

### 어드민 세션 규칙
- **F5 새로고침**: 어드민 화면 유지
- **새 URL 입력 / 새 탭**: 어드민 세션 초기화 → 일반 화면으로 진입

### 사용자 관리
- 등록된 환자 목록 확인
- 환자 이름 터치 → 환자 상세 정보 확인 (기본정보, PSA, 증상, 복약, 식단 요약)
- **환자 화면으로 보기**: 해당 환자가 보는 것과 동일한 화면으로 미리보기
- 환자 삭제

### 환자 미리보기
- 환자 상세 화면 → "○○○ 님 화면으로 보기" 버튼 터치
- 화면 하단에 **🛡️ 어드민 미리보기 중** 배너 표시
- **복귀** 버튼 터치 → 어드민 환자 상세 화면으로 복귀

### 환자 현황 모니터링
- 오늘 기록 현황 (식사·운동·증상) 전체 목록 확인

### 백업 / 복원
- 전체 사용자 데이터 텍스트 백업
- 백업 데이터 붙여넣기로 복원

## 운영 주의사항

- 이 앱은 **의료 행위가 아닙니다**. 치료·처방 표현은 사용하지 않습니다.
- 배포 시 반드시 `sw.js`의 CACHE 버전을 올려야 사용자 기기에 업데이트가 적용됩니다.
- 어드민 비밀번호는 별도 관리하세요.
